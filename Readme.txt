- Seleccionar placa "AI Thinker ESP32-CAM" para el módulo de la cámara.
- Seleccionar placa "ESP32 Dev Module" para el módulo con el sensor.

- Añadir librería ESP32 en el IDE de Arduino: 
	- En Archivo->Preferencias...
	- Añadir URL "https://dl.espressif.com/dl/package_esp32_index.json"


- Conexiones sensor PIR:
	-    +   <-> Vin
	- output <-> D33
	-    -   <-> GND
